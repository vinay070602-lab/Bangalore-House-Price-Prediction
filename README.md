# Bangalore-House-Price-Prediction
Machine learning project predicting Bangalore house prices using regression models. Includes data cleaning, feature engineering, outlier removal, and model evaluation. Built with Python, Pandas, NumPy, and Scikit-learn to analyze housing data and generate accurate price predictions.

## Project Overview
The goal of this project is to estimate property prices based on factors such as location, size, number of bedrooms, and other housing features.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Steps Performed
1. Data cleaning and preprocessing
2. Feature engineering
3. Exploratory Data Analysis (EDA)
4. Model training using regression algorithms
5. Model evaluation

## Machine Learning Model
- Linear Regression
